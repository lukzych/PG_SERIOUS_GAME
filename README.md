# PG_SERIOUS_GAME

Aolikacja desktopowa napisana w Java Swing, wspierająca rehabilitację kognitywno-motoryczną. Zawiera mini-gry angażukące orientację, koncentrację i pamięć krótkotrwałą - sprawiająć że terapia staje się interaktywna i mierzalna
